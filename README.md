# magic-link-authentication
Passwordless authentication system based on one-time magic login links

This project implements a magic link authentication system. Users can log in without a password by using a magic link sent to their email.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/modestick/magic-link-authentication.git
   ```
2. Navigate to the backend directory:
   ```bash
   cd magic-link-authentication/backend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open your browser and navigate to `http://localhost:3000`.
